1. X-Ray
  Інструмент для швидкого пошуку кандидатів у LinkedIn.
  Генерує спеціальне посилання за заданими параметрами (позиція, країна, ключові слова), що дозволяє миттєво відкрити відфільтрований пошук вакансій або профілів людей.

2. Проєкти 
   Модуль інтегрується з Google Sheets та виконує дві ключові дії:
     Виводить список доступних проєктів із таблиці.
     Приймає посилання на веб-сторінку → парсить сайт → дістає потрібну інформацію → автоматично записує дані у Google Sheets.
   Це спрощує збір даних про проєкти.
   
{в розробці}3. Постик
  Після отримання посилання на проєкт:
    Збирає всю необхідну інформацію зі сторінки.
    Формує структурований та стильний пост.
    Готовий пост одразу публікує у Telegram-канал.
  
4. рМкалендар
   Модуль автоматизує роботу з календарем:
     Зчитує дані з Google Sheets.
     Витягує потрібні параметри (дати, назви).
     Створює .json з підготовленими подіями.
     На основі цього JSON автоматично додає всі події в Google Calendar.
